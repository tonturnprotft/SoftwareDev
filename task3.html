Guide for Cross-Compiling C Code for Raspberry Pi 4B on Ubuntu using VS Code

Setup: Install Cross-Compilation Tools on Ubuntu

  1. Install the required cross-compilation tools.

    Install the GCC toolchain for ARM architecture using the command: sudo apt install gcc-arm-linux-gnueabihf

  2. Verify the installation.

    Run arm-linux-gnueabihf-gcc --version to ensure the cross-compiler is installed correctly.

Task 1: Write and Cross-Compile C Code using VS Code

  1. Write a simple C program (main.c).

    Example C code for Raspberry Pi: #include <stdio.h> int main() { printf("Hello from Raspberry Pi!"); return 0; }

  2. Set up tasks.json for VS Code.

    Create a build task in VS Code to use the ARM cross-compiler for building the C program.

  3. Compile the C program.

    Use the build task in VS Code (Ctrl+Shift+B) to compile the C code for the Raspberry Pi.

Task 2: Transfer and Execute the Program on Raspberry Pi

  1. Transfer the executable using scp.

    Use the command: scp main_rpi pi@:/home/pi/ to transfer the compiled program.

  2. Execute the program on Raspberry Pi.

    SSH into your Raspberry Pi and run the executable with ./main_rpi. The output should display "Hello from Raspberry Pi!"

Troubleshooting

  • Issue: Cannot find the Raspberry Pi on the network.
    Solution: Check your Raspberry Pi's IP using the command ifconfig and ensure SSH is enabled.
  • Issue: Cross-compilation errors.
    Solution: Ensure the correct toolchain is installed and verify the architecture using file main_rpi.
