Using VS Code with SSH to Compile and Run C Programs on Raspberry Pi

Setup: Connect VS Code to Raspberry Pi using SSH

  1. Install VS Code Extensions.

    Ensure you have installed the following extensions:

    • Remote - SSH: For connecting to your Raspberry Pi.
    • C/C++: For syntax highlighting and IntelliSense for C/C++ code.

  2. Connect to Raspberry Pi via SSH.

    Open VS Code, press Ctrl+Shift+P, and select Remote-SSH: Connect to Host. Enter the SSH connection string, e.g., pi@.

Task 1: Write and Compile C Code on Raspberry Pi

  1. Create a C program on Raspberry Pi.

    Inside VS Code, create a file named hello.c on the Raspberry Pi:

    #include <stdio.h>
int main() {
    printf("Hello, World!\\n");
    return 0;
}

    This is a simple "Hello World" C program that will print the message to the terminal.

    Here is what the code should look like in VS Code:

  2. Compile the program on Raspberry Pi.

    Open the integrated terminal in VS Code (using the SSH connection to Raspberry Pi) and run the following command to compile the C file using GCC:

    gcc hello.c -o hello
  3. Run the compiled program.

    Once compiled, run the program directly in the terminal:

    ./hello

    This should output: Hello World!

Troubleshooting

  • Issue: SSH connection fails.
    Solution: Ensure SSH is enabled on Raspberry Pi by running sudo raspi-config and selecting Interfacing Options > SSH.
  • Issue: Compilation errors.
    Solution: Verify that GCC is installed on Raspberry Pi by running gcc --version. If not installed, use sudo apt install gcc.

VNC Remote Desktop Setup Guide for Raspberry Pi 4B

Remote Desktop Access using VNC for Raspberry Pi 4B

Task: Set up and Access VNC Remote Desktop

  1. Enable VNC on Raspberry Pi Desktop (Raspberry Pi OS).

    Open the Raspberry Pi Configuration tool, go to the 'Interfaces' tab, and enable VNC. Then, start the VNC server with the command: sudo systemctl start vncserver-x11-serviced.

  2. Enable VNC on Ubuntu Desktop.

    Install the VNC server with sudo apt install tightvncserver. Enable Screen Sharing via Settings > Sharing.

  3. Install VNC Viewer on Windows 10/11.

    Visit the RealVNC website to download VNC Viewer. Install the software on your Windows machine.

  4. Find Raspberry Pi's IP Address.

    On your Raspberry Pi, open a terminal and run ifconfig. Find the IP address under the network interface labeled 'inet'.

  5. Connect to Raspberry Pi using VNC Viewer.

    Open VNC Viewer on Windows and enter the Raspberry Pi's IP address (e.g., 192.168.1.100:1). Log in with your Raspberry Pi's credentials.

  6. Access Raspberry Pi Desktop via VNC.

    Once connected, you should be able to see the Raspberry Pi desktop. The default desktop environment on Raspberry Pi OS uses X11.

    Screenshot of Raspberry Pi desktop accessed via VNC.

  7. Switch to Wayland Desktop (Ubuntu only).
    1. Log out from your Ubuntu Desktop session.
    2. On the login screen, click the gear icon and select Ubuntu on Wayland.
    3. Log in and reconnect via VNC.

Troubleshooting

  • Issue: Cannot connect via VNC.
    Solution: Verify VNC is enabled and check if the correct IP address is used.
  • Issue: Wayland session is slow or unresponsive.
    Solution: Try using the X11 desktop environment, or update the VNC server and client to the latest version.
